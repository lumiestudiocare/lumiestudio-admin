# 🦋 Lumiê Studio — Painel Administrativo

Painel admin completo com Supabase Realtime.

## Stack
- React 19 + TypeScript + Vite
- Supabase (PostgreSQL + Auth + Realtime)
- Zustand, Recharts, Font Awesome
- GitHub Pages + CI/CD automático

## Setup

```bash
npm install
# Crie .env.local com:
# VITE_SUPABASE_URL=https://xxx.supabase.co
# VITE_SUPABASE_ANON=eyJ...
npm run dev
```

## Acesso
URL: https://admin.lumiestudio.com.br  
Auth: Supabase Auth (configure o usuário no dashboard do Supabase)
